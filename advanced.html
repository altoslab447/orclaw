SESSION MANAGEMENT

用 Session 同时处理多件事

Session 是大闸蟹的核心概念，让你像开多个浏览器分页一样，让 AI 同时处理不同任务。

?

什么是 Session？

每个 Session 是一个独立的对话上下文。

Session A
株式监控
持续监控 TSLA、2330 即时报价
Active
Session B
日常问答
翻译、搜寻、一般对话
Active
Session C
资料整理
长时间资料分析任务（执行中）
Running
⇄
互不干扰
各 Session 完全独立
⚡
同时并行
多任务同步执行
◉
独立记忆
各自有专属上下文

如何使用 Session

01

建立新 Session

/session new 股票监控

→ 建立一个名为「股票监控」的 Session 

/session new 日常助理

→ 再建立一个「日常助理」Session

02

切换 Session

/session switch 股票监控

→ 切换到「股票监控」Session，之后的对话都在这个 Session 进行 

/session list

→ 列出所有 Session 及其状态

03

删除 Session

/session delete 股票监控

! 删除 Session 会同时清除该 Session 的所有对话记忆，无法复原。重要的 Session 请谨慎删除。

实际应用范例

FINANCE + DAILY
财经监控 + 日常助理分离
SESSION "财经"
设定每日财经早报排程、股票提醒
SESSION "助理"
日常问答、翻译、摘要任务
两者互不影响，财经排程不会被日常对话打断
LONG TASK + INSTANT
长任务 + 即时查询
SESSION "长任务"
整理大量资料、分析报告（可能需要数分钟）
SESSION "即时"
同时进行其他查询，不需等待长任务完成
不再卡在等待，多工并行
MULTI-PROJECT
多专案管理
SESSION "专案A"
追踪 A 专案相关资料、待办
SESSION "专案B"
追踪 B 专案，各自有独立记忆
就像有多个专属助理，各司其职

→ Session 使用技巧

  • — Session 名称建议有意义（用途/专案名），方便记忆
  • — 重要 Session 不要随意删除，记忆会一起消失
  • — 切换 Session 前可以先 /session list 确认有哪些
  • — 排程任务建议放在独立 Session，避免被日常对话干扰
/02
SKILL MANAGEMENT

透过 ClaWHub 安装技能

Skills 是大闸蟹的插件系统，让你无限扩充 AI 的能力。

🏪

什么是 ClaWHub？

ClaWHub 是大闸蟹的技能市集，类似 App Store。你可以在 Telegram 对话中直接安装、移除、更新技能，无需登入任何后台。

三种安装方式

1

直接告诉 AI（最简单）

最直观的方式：直接用自然语言告诉大闸蟹你想要什么，它会自动帮你执行。

你： 帮我安装 stock-info-explorer 技能
🦀： 好的，正在为你安装 stock-info-explorer... 安装完成！现在你可以问我股票价格了。

i AI 会自动执行安装并确认结果。如果有错误，也会自动尝试修复。

2

直接在伺服器执行指令

进阶用户可以透过 SSH 直接在伺服器执行 clawhub 指令。

bash 
# 安装单一技能
clawhub install SKILL_NAME

# 安装需要强制安装的技能（VirusTotal 误判）
clawhub install --force SKILL_NAME

# 移除技能
clawhub remove SKILL_NAME

# 更新所有技能
clawhub update

# 列出已安装技能
clawhub list

# 搜寻可用技能
clawhub search KEYWORD

! 部分技能需要加上 --force 参数，这是因为 VirusTotal 对某些技能有误判，并非真的有安全问题。

3

让 AI 自建技能（skill-creator）

安装 skill-creator 技能后，你可以告诉 AI 你想要什么功能，它会自动开发、测试并安装。

你： 帮我做一个每天爬取 PTT 科技板热帖、整理 5 条重点的技能
🦀： 好的，我来帮你建立这个技能，正在分析 PTT 结构...
[AI 自动写程式、测试、安装]
🦀： 技能「ptt-tech-summary」已安装，每天会自动执行，并将重点摘要传送给你。

✦ skill-creator 让大闸蟹的能力真正无限。只要你想得到，都可以让它帮你开发。

RECOMMENDED SKILLS

推荐技能清单

以下是精选的实用技能，覆盖日常使用的核心需求

🔍

ddg-web-search

搜寻

DuckDuckGo 即时搜寻，不需要任何 API Key。询问最新资讯时自动触发。

clawhub install ddg-web-search
免费 无需 API Key ★ 预装推荐
📈

stock-info-explorer

金融

美股、A股即时报价。支援 TSLA、贵州茅台 (600519)、ETF 等，基于 Yahoo Finance。

clawhub install --force stock-info-explorer
免费 即时数据 需 uv
₿

crypto-market-data

金融

主流加密货币即时行情，BTC、ETH、SOL 等，含 24h 涨跌幅。

clawhub install --force crypto-market-data
免费 即时数据
📰

news-summary

资讯

指定主题的今日新闻摘要，AI 自动搜寻并整理成重点列表。

clawhub install --force news-summary
支援任意主题 繁中支援
🎬

youtube-transcript

工具

贴入 YouTube 网址，自动撷取字幕并整理重点，省时省力。

clawhub install youtube-transcript
免费 支援多语言字幕
🌐

translate

语言

多语言互译，中英日韩等，AI 驱动比传统翻译更自然流畅。

clawhub install translate
免费 AI 驱动
📄

summarize

工具

长文摘要整理，贴入文章、报告、合约，快速提取重点。

clawhub install summarize
免费 支援超长文本
🌤️

weather

生活

全球城市即时天气查询，基于 Open-Meteo，不需要任何 API Key。

clawhub install weather
免费 无需 API Key ★ 预装推荐
🛠️

skill-creator

进阶

让 AI 帮你自动开发新技能。告诉 AI 你的需求，它写程式、测试、安装，一条龙完成。

clawhub install --force skill-creator
无限扩充 自然语言操作
